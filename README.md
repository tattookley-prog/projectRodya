# Random Quote Generator — projectRodya

**Автор:** Галимов Родион Фирдависович

---

## Описание проекта

«Random Quote Generator» — это GUI-приложение на Python (tkinter), которое:

- хранит коллекцию из 20 предопределённых цитат (текст, автор, тема);
- генерирует случайную цитату по нажатию одной кнопки;
- позволяет фильтровать цитаты по автору и теме;
- ведёт историю показанных цитат и сохраняет её в `history.json`;
- позволяет добавлять собственные цитаты прямо из интерфейса.

---

## Требования

- Python 3.8+
- `tkinter` — встроен в стандартную поставку Python (дополнительной установки не требует)

---

## Как запустить

```bash
python main.py
```

---

## Примеры использования

| Действие | Описание |
|---|---|
| Нажать «Сгенерировать цитату» | В центре окна появится случайная цитата из пула |
| Выбрать автора в фильтре | Следующая генерация выберет цитату только этого автора |
| Выбрать тему в фильтре | Аналогично, но по теме |
| Нажать «Сбросить фильтры» | Фильтры возвращаются в «Все» |
| Заполнить форму и нажать «Добавить» | Новая цитата появляется в пуле |
| Оставить поле пустым и нажать «Добавить» | Появится сообщение об ошибке |
| Закрыть и снова открыть приложение | История цитат восстанавливается из `history.json` |

## Возможные тесты (ручная проверка)

1. **Генерация без фильтров** — нажать кнопку несколько раз, убедиться, что цитаты меняются.
2. **Фильтр по автору** — выбрать конкретного автора, убедиться, что все сгенерированные цитаты принадлежат ему.
3. **Фильтр по теме** — аналогично с темой.
4. **Пустой пул по фильтрам** — установить несовместимые фильтры (автор А + тема Б, которых нет вместе), убедиться, что появляется предупреждение.
5. **Добавление цитаты** — заполнить все три поля, нажать «Добавить», убедиться, что цитата появляется в выпадающих списках.
6. **Валидация** — нажать «Добавить» с пустым полем, убедиться, что появляется messagebox с ошибкой.
7. **Сохранение истории** — сгенерировать несколько цитат, закрыть и снова открыть приложение, убедиться, что история восстановилась.
8. **Сброс фильтров** — установить фильтры, нажать «Сбросить фильтры», убедиться, что оба поля стали «Все».
